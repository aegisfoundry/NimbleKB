## Symptoms
- Any user that logs into the system is forced to log in, even if they are a manager.
- After clocking in, any user trying to create an order is presented with a PIN Pad asking them for a user with permission to let them create an order. No users permissions for that work.
- No users can get to the back office from the front of house interface, including managers.

## Resolution
Ensure that your system is configured to use US standards on date and times. Some international settings have been found to cause issues.
