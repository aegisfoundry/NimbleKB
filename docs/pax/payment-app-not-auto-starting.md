# Symptom

The payment app (Tsys, BroadPOS, etc) is not running at the start of day

# Resolution

The payment device restarts nightly for compliance reasons.

If the payment application is not automatically starting, ensure the setting "Dedicated Device: Full Screen Mode" under Misc -> General Settings is enabled.

Please see the the article on pax configuration for detail of required / recommended settings for Pax devices.
