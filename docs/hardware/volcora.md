# Volcora point-of-sale Terminals

## Hardware
Varies. 
Typically Core i5-4200 or similar CPU
RAM 4-8 Gig
Hard drive 64-256 gig
Screen Resolution: 1368xsomething or other OR 1920x1080

The power button is typically circular press buttons on the back side of the bottom-right corner of the screen.

## Operating System
Typically Windows 11 Pro

### Note On Windows Activation
Volcora ships their computers without a valid Windows license. You will need to request the license key from Volcora using the follwing link:
https://volcora.com/pages/volcora-terminal-windows-license-key-request

You will need the device serial number, located on a sticker on the back of the unit behind the network ports. It is a very long S/N, ~ 28 digits long. Of note, the zeros are slashed and they do use capital "O"s in their S/N range.
You will also need the order ID from Volcora.

Once you fill out the link, they will email you the windows key in a few moments. To activate, go to Windows Activation on the device, select change the product key, and enter the one you were emailed.
